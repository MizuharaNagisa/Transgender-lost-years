# 本文将阐述如何正确使用激光进行脱毛
## 设备推荐：
Tria4x或其他每平方厘米能量超过7J的激光器，最低波长808nm <br>
冠舟typelight，IPL，每平方厘米能量7J，新版7.5J，并有快速出光的国内 <br>

## 麻醉
由于激光功率本身过大，因此需要进行麻醉作为疼痛管理
### 麻醉药物及使用
利多卡因凝胶，使用方法为敷于要打激光的部位半小时 <br>
EMLA凝胶，为利多卡因2.5%和丙胺卡因2.5%的混合物，处方药，需于医师指导下使用，使用方法为敷于要打激光的部位半小时 <br>
